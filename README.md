# Project_Training_Planner_Evgenii
Создатель: Бессмертных Евгений
Описание: Это приложение предназначено для организации ваших тренировок. Вы можете добавлять новые тренировки, указывая дату, тип и продолжительность. Также есть возможность искать конкретные тренировки по дате или типу.
Инструкция по использованию:
Запустите программу.
Введите дату тренировки (например, 2023-08-15), укажите тип упражнения (например, бег) и продолжительность в минутах.
Нажмите кнопку «Добавить тренировку», чтобы сохранить запись.
Для поиска определенных тренировок введите нужный тип или дату в соответствующие поля и нажмите «Применить фильтр».
Чтобы просмотреть все сохраненные тренировки, нажмите «Сбросить фильтр».
Все введенные данные будут сохранены в файле trainings.json, чтобы информация не исчезла при закрытии программы.
